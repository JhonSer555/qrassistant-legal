Политика конфиденциальности

QRAssistant — обработка и защита данных пользователей

Настоящая Политика конфиденциальности описывает порядок использования информации при работе приложения QRAssistant.

1. Общие положения

Приложение QRAssistant предназначено для создания и сканирования QR-кодов. В процессе работы используется только минимально необходимая информация для обеспечения функциональности.

2. Какие данные могут использоваться

  • технические идентификаторы устройства или пользователя;
  • данные, необходимые для работы функций приложения;
  • данные, связанные с использованием QR-функций.

3. Использование данных

Данные используются исключительно для работы приложения, улучшения стабильности и предоставления функционала.

4. Передача данных третьим лицам

Данные не продаются и не передаются третьим лицам, кроме случаев, предусмотренных законодательством РФ.

5. Безопасность

Применяются разумные технические меры для защиты данных пользователей.
