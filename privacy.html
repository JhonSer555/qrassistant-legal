Политика конфиденциальности

QRAssistant — обработка и защита данных пользователей

Настоящая Политика конфиденциальности описывает порядок обработки данных при использовании приложения QRAssistant.

1. Какие данные могут обрабатываться

Приложение может обрабатывать:

  • технические идентификаторы устройства;
  • данные лицензии PRO;
  • Recovery Key / Recovery QR;
  • информацию о платежах;
  • технические данные, необходимые для работы приложения.

2. Платежи

Обработка платежей осуществляется через сторонние платежные сервисы, включая Robokassa.

3. Хранение данных

Для хранения технических данных и лицензий может использоваться платформа Supabase.

4. Передача данных третьим лицам

Данные не продаются и не передаются третьим лицам, за исключением случаев, необходимых для обработки платежей или предусмотренных законодательством РФ.

5. Безопасность

Принимаются разумные технические меры для защиты данных пользователей.
