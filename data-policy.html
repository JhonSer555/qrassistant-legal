Политика обработки персональных данных

1. Общие положения

Настоящая Политика определяет порядок обработки персональных данных пользователей приложения QRAssistant PRO.

Оператором персональных данных является самозанятый Фадеев Дмитрий Витальевич.

2. Какие данные обрабатываются

  • уникальный идентификатор пользователя (например, QRA-XXXXXX);
  • информация о статусе оплаты и активации PRO-доступа;
  • данные, необходимые для работы сервиса Supabase.

Приложение не требует указания паспортных данных или избыточной персональной информации.

3. Цели обработки данных

  • предоставление доступа к функционалу QRAssistant PRO;
  • обработка и подтверждение оплаты;
  • обеспечение работы приложения.

4. Передача третьим лицам

Обработка платежей осуществляется через платёжную систему Robokassa.

Оператор не получает данные банковских карт пользователей.

5. Хранение данных

Данные хранятся в базе Supabase и используются исключительно для функционирования приложения.

6. Права пользователя

Пользователь имеет право запросить удаление своих данных или ограничение их обработки.
